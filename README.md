### 🦞 AivoClaw 解决了什么问题？

> **你不应该为了用一个 AI 工具，先学会配环境。**
> 安装 → 填 Key → 干活。三步搞定，无需技术背景。

[OpenClaw](https://github.com/openclaw/openclaw) 是 2026 年 GitHub 上增长最快的开源 AI 智能体项目，但它依赖 Node.js 22+、命令行操作和一系列手动配置——对普通用户来说门槛太高。

AivoClaw 做的事情很简单：**帮你跳过所有技术准备工作**。我们把 Node.js 运行时、OpenClaw Gateway 服务全部打包进一个安装程序，双击就能用。

### 📦 获取安装包

访问 [Releases 页面](https://github.com/yuanyuekejiJN/AivoClaw/releases/latest)，选择你的平台下载：

| 平台 | 芯片架构 | 文件名 |
|---|---|---|
| 🍎 macOS | Apple Silicon (M1/M2/M3/M4) | `AivoClaw-x.x.x-arm64.dmg` |
| 🍎 macOS | Intel | `AivoClaw-x.x.x-x64.dmg` |
| 🪟 Windows | x64 | `AivoClaw-Setup-x.x.x-x64.exe` |
| 🪟 Windows | ARM64 | `AivoClaw-Setup-x.x.x-arm64.exe` |

> **怎么选？** M 系列芯片的 Mac 下载 arm64 版本，老款 Intel Mac 下载 x64 版本，Windows 用户大多数情况选 x64 即可。

### 🚀 快速开始

```
1.  运行安装程序（macOS 拖入应用文件夹 / Windows 点击安装向导）
2.  选择 AI 服务商，粘贴你的 API Key
3.  输入你的第一条指令，开始体验
```

全程无需安装 Node.js、npm 或任何开发依赖。

### ❓ 常见疑问

**Q: 不懂编程也能用吗？**
A: 完全没问题。AivoClaw 的设计初衷就是降低 AI 工具的使用门槛。

**Q: 需要提前安装 Node.js 或其他开发工具吗？**
A: 不需要，所有依赖已经打包在应用内部。

**Q: 首次设置后还能更换 AI 服务商吗？**
A: 随时可以。通过系统托盘菜单进入「设置」即可切换（macOS 也可使用 `Cmd+,` 快捷键）。

**Q: 飞书集成怎么用？**
A: 配置飞书 Webhook 后，AivoClaw 可以作为你飞书工作区里的智能助手，直接在群聊中响应指令。

应用在用户目录下创建以下文件：

```
~/.openclaw/
├── openclaw.json                    用户配置（模型、令牌、频道等）
├── aivoclaw.config.json             AivoClaw 标识文件（设备 ID、安装时间）
├── openclaw.last-known-good.json    最近一次成功启动的配置快照
├── app.log                          应用日志
├── gateway.log                      Gateway 子进程日志
├── config-backups/                  配置滚动备份（最多 10 份）
├── credentials/                     凭据存储
│   └── kimi-search-api-key          Kimi Search API Key
├── workspace/skills/                已安装的社区技能
└── bin/
    └── openclaw                     CLI 命令包装脚本
```
